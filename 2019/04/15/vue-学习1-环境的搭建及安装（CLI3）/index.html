1. 安装node环境和npm

Node 版本要求
Vue CLI 需要 Node.js 8.9 或更高版本 (推荐 8.11.0+)。

检查node版本及npm版本

2. 安装cli3

Vue CLI 的包名称由 vue-cli 改成了 @vue/cli。 如果已经全局安装了旧版本的 vue-cli (1.x 或 2.x)，先通过 npm uninstall vue-cli -g 或 yarn global remove vue-cli 卸载它。

安装完后使用 vue -V / vue –version 检查是否安装正确。
